A collection of backend development practice projects and tutorials built while learning server-side development concepts, APIs, databases, and deployment workflows.

## 📂 Repository Structure

```bash
Back-end-development/
│

│
```

## 🚀 Projects Included

### 1️⃣ Backend Tutorial

This folder contains backend learning materials, practice projects, and experiments while exploring:

- Node.js
- Express.js
- REST APIs
- Routing
- Middleware
- MongoDB
- CRUD Operations
- Authentication Basics

#### Purpose
- Strengthen backend fundamentals
- Practice API creation and server-side logic
- Understand client-server architecture

### 2️⃣ Restaurant Website

A backend-powered restaurant website project with features such as:

- User Authentication
- Signup/Login System
- Menu Management
- Order Handling
- Database Connectivity
- API Integration
- JWT Authentication
- Role-based Authorization

#### Tech Stack
- Node.js
- Express.js
- MongoDB
- REST API
- Git & GitHub

#### Purpose
- Build a real-world backend project
- Learn deployment workflows
- Practice full-stack integration
